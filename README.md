  Built With
  </h2>
<ul dir="auto">
  <li><a href="https://jdk.java.net/17/" rel="nofollow">Java</a> - The programming language used for the back-end development.</li>
  <li><a href="https://openjfx.io/" rel="nofollow">JavaFX</a> - The programming language used for the front-end development.</li>
  <li><a href="https://www.w3schools.com/css/" rel="nofollow">CSS</a> - The programming language used for the front-end development.</li>
